# ChrysaLisp

C++ ChrysaLisp !

A version of ChrysaLisp Lisp written in C++.

Needs a few OS specific missing built in functions from ChrysaLisp yet. However
it can now use the same compiler enviroment from ChrysaLisp OS to build a full
OS boot image !


Make with:

```
make -j
```

Clean with:

```
make clean
```

Run with:

```
./chrysalisp
```
